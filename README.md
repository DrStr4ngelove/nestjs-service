# CREATE YOUR OWN NINJA ARMY!!
This small service will utilize API routing to create, update, and delete ninjas for your personal army using the NestJS framework.

## Install

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
